# Configuration de l’environnement

## Version Python

Le projet est standardisé sur :

Python 3.12.9

Tous les membres de l’équipe doivent utiliser cette version afin de garantir la compatibilité des dépendances et la reproductibilité de l’environnement de travail.

## Mise en place

```bash
python --version
